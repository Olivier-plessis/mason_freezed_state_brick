## using mason
 create mason bricks
 ```mason new brick_name -o bricks```
add brick to mason.yaml
```mason add brick_name --path bricks/brick_name```

and start to project folder
```mason get```
```mason add brick_name```
to remove a brick 
```mason remove brick_name```
